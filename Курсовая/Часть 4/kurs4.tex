\section*{Вычисление ОИКО ядерной оценки методом Монте-Карло и оценка закона его распределения}

В четвёртой части курсовой работы проводится численное исследование точности ядерной оценки плотности равномерного распределения с помощью метода Монте-Карло, а также оценивается закон распределения величины ОИКО.

Рассматривается равномерное распределение с плотностью:
\[
f_0(x) = 
\begin{cases}
\dfrac{1}{2\sqrt{3}}, & |x| \le \sqrt{3}, \\
0, & |x| > \sqrt{3}.
\end{cases}
\]

Ядерная оценка плотности с колокольным ядром имеет вид:
\[
\hat{f}_h(x) = \frac{1}{nh} \sum_{i=1}^n K\left( \frac{x - X_i}{h} \right),
\]
где \( K(u) = \frac{1}{\sqrt{2\pi}} \exp\left( -\frac{u^2}{2} \right) \), а параметр сглаживания \(\xi = h\).

Нормирующий множитель (квадрат нормы теоретической плотности):
\[
\int_{-\infty}^{\infty} f_0^2(x)\, dx = \frac{1}{2\sqrt{3}} = \frac{1}{L}, \quad L = 2\sqrt{3}.
\]

\subsection*{1. Вычисление ОИКО методом Монте-Карло (ММК)}

Для получения относительной интегральной квадратической ошибки (ОИКО) выполняются следующие шаги:

1. Генерируется \(N\) независимых выборок объёма \(n\) из распределения \(f_0(x)\).


2. Для каждой выборки и каждого значения \(\xi\) строится ядерная оценка \(\hat{f}(x)\).


3. Вычисляется интегральная квадратическая ошибка (ISE):
   \[
   \text{ISE}^{(k)}(\xi) = \int_{-\infty}^{\infty} \bigl(\hat{f}^{(k)}(x) - f_0(x)\bigr)^2 \, dx.
   \]
   
4. ОИКО получается нормировкой на \(\int f_0^2(x)\, dx\):
   \[
   \bar{\delta}_n(\xi) \approx \frac{1}{N} \sum_{k=1}^N \frac{\text{ISE}^{(k)}(\xi)}{\int f_0^2(x)\, dx} = L \cdot \frac{1}{N} \sum_{k=1}^N \text{ISE}^{(k)}(\xi).
   \]

\subsection*{2. Оценка закона распределения величины ОИКО}

Фиксируется объём выборки \(n\) и оптимальное значение параметра \(\xi\). Генерируется \(N\) независимых реализаций, для каждой из которых вычисляется значение \(\bar{\delta}_n^{(k)}(\xi)\).

Полученная выборка \(\{\bar{\delta}_n^{(k)}\}_{k=1}^N\) используется для построения эмпирической гистограммы закона распределения ОИКО ядерной оценки.
